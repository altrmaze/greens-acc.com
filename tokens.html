Active Tokens

Enter your User ID and click Load Tokens to view your API tokens.

Using your API token

Pass your token in the Authorization header on every API request:

curl -X POST https://your-project.supabase.co/functions/v1/manageApiTokens \
  -H "Authorization: ******" \
  -H "Content-Type: application/json" \
  -d '{"action":"list","user_id":"<your-uuid>"}'

Available scopes:

read:deals write:deals read:shipments write:shipments read:compliance read:marketplace write:marketplace admin
